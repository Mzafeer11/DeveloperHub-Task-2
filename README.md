# DeveloperHub Task 2: Stock Prediction

This repository contains the solution for Task 2 of the DeveloperHub Internship program.

## Project Overview
This project focuses on time-series analysis and forecasting. The goal is to predict future stock prices based on historical market data using machine learning techniques. 

## Files
- `stock-prediction.ipynb`: The Jupyter Notebook containing the data ingestion, exploratory data analysis, feature engineering, and model prediction code.

## How to Run
1. Ensure Python and Jupyter are installed.
2. Install necessary dependencies (e.g., pandas, scikit-learn, matplotlib).
3. Run the notebook cells to train the forecasting model and view the prediction graphs.
